Pour créer votre propre CV en ligne
il y a deux façon de le faire

****************méthode 1 :****************
vous avez besoin d'un compte github + vscode

etape 1 :
créer un dossier local sur le bureau appelé par exemple GITHUB
ensuite ouvre ton vscode et ouvre le dossier que tu viens juste de créer (càd GITHUB)
dedans vous pouvez par exemple suivre cette structure :
[GITHUB
    docs
        documentation
        pictures
    index.html
    README.md]
dans le fichier index.html vous devez le remplir par le code de votre CV
dans le fichier README.md vous devez le remplir par expliquer à quoi ça sert votre projet (dans notre cas cest d'avoir un CV en ligne)
dans le dossier docs on peut mettre un ensemble de fichiers/dossiers supplémentaires
dans le dichier documentation cest où on peut mettre d'autres documentations qui nous sert et aide les autres à mieux comprendre le projet (plus détaillé que le fichier README.md)
dans le dossier pictures on peut mettre tout les images du projet
